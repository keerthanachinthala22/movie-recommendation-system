# Movie Recommendation System
## Description
This project is a Content-Based Movie Recommendation System built using Python, Scikit-learn, TF-IDF Vectorizer, Cosine Similarity, and Streamlit.
## Dataset
TMDB 5000 Movies Dataset
## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
## Run the Project
streamlit run task6.py
